# Grid-Editor
Simple Grid Editor in Unity

- Camera Controller
  - WASD movement
  - Look around (RIGHT-CLICK)
  - Zoom in and out
 
- Object actions
  - Spawn (SPACE)
  - Select (LEFT-CLICK)
  - Deselect (ESCAPE)
  - Delete (BACKSPACE)
  - Type and Position are saved after application quit
- Raycasting usage
- Slider to change grid size and length in runtime
